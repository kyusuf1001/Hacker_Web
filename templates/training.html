Training — Rules & Commands

Wires

  1. if 2 lights and wires are red & blue → command: connect red blue
  2. if 2 lights and wires are green & yellow → command: cut green
  3. if 3 lights (any pair shown) → command: disconnect all
  4. if 1 light with red and another color → command: cut <color> (not red)

Keypad

  1. if number is even → pass = number * 2
  2. if number is odd → pass = number + 3
  3. if number is 9 → pass = 999

Firewall

Pattern starts with A / B / C / D:

  • A → type the pattern backwards
  • B → type the pattern twice (repeat)
  • C → drop the middle character
  • D → type the pattern as-is

Use these commands exactly in the hack console.
